## هيكل الملفات لكل نشاط جديد

```
js/activities-catalog.js       ← تعريف النشاط + serviceIds + booking
js/services-catalog.js         ← الخدمات (activityId, slotDuration, requiresAddress)
js/content-templates/{id}.js   ← محتوى الواجهة
js/ui-profiles/registry.js     ← تسجيل المحتوى
index.html / admin.html / book.html ← script tag
data/config.json               ← enabledActivities + enabled
```
